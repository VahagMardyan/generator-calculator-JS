Here are some functions for calculator.


  • For π type: PI
  • For e type: E
  • For sin(argument) type: sin(argument)
  • For cos(argument) type: cos(argument)
  • For tg(argument) type: tan(argument)
  • For ctg(argument) type: 1/tan(argument)
  • For logbaseexponent type: log(exponent)/log(base)
  • For absolute value type: abs(argument)
  • For square root (√argument) type: sqrt(argument)
  • For baseexponent type: pow(base,exponent)
  • For cubic root type: cbrt(argument)
  • For eargument type: exp(argument)
  • For round up type: ceil(argument) (ex. ceil(5.4)=6)
  • For round down type: floor(argument) (ex. floor(5.4)=5)
  • For round type: round(argument) (ex. round(4.23)=4, round(6.5)=7)
